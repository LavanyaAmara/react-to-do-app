# React Todo App 📝

A simple and interactive To-Do List app built using **React**.  
You can add, delete, move up, and move down your tasks easily.

## 🚀 Features
- Add new tasks  
- Delete existing tasks  
- Move tasks up or down in the list  
- Clean and responsive UI  

## 🛠️ Technologies Used
- React (useState Hook)
- HTML, CSS, JavaScript
